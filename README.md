📋 Projecto- Task Management Android App

A modern Android task management application designed to help users organize, track, and manage daily tasks efficiently. The app provides an intuitive interface for creating, updating, and monitoring tasks to improve productivity and workflow management.

✨ Features

Create, update, and delete tasks easily

Organize tasks with clear task lists

User-friendly and responsive Android UI

Efficient task tracking for daily productivity

🛠️ Technologies Used

Java / Kotlin

Android Studio

SQLite / Room Database

Material Design UI Components
